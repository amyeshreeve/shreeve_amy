# Purpose
This repository is for RHE 330C: Rhetorical Metrics with Dr. S. Scott Graham at UT Austin.
Find more information on this course [here](https://liberalarts.utexas.edu/rhetoric/faculty/ssg2394#courses).

* Framegrams
